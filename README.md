 Project Description
This project contains an automated test suite for the Singlish-to-Sinhala transliteration engine using Playwright. It covers 24 positive functional cases, 10 negative robustness cases, and real-time UI validation.

 Prerequisites
- Node.js installed
- Playwright installed

 Execution Instructions
         Clone the repository:
           `git clone [Your Repository Link]`
         Install dependencies:
            `npm install`
         Run the tests:
            `npx playwright test`
          View Report:
            `npx playwright show-report`
